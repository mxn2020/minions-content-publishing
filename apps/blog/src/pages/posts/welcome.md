# Welcome to Minions Content-publishing

We're excited to introduce **Minions Content-publishing** — Publish queue, delivery receipts, and mode configuration.

## What's Included

- **TypeScript SDK**: `npm install @minions-content-publishing/sdk`
- **Python SDK**: `pip install minions-content-publishing`
- **CLI**: `npm install -g @minions-content-publishing/cli`

## Getting Started

Check out the [documentation](https://content-publishing.minions.help) for a full guide.
