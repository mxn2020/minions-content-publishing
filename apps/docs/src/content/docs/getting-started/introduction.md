# Minions Content-publishing

Publish queue, delivery receipts, and mode configuration

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-content-publishing/sdk`) — Core library for Node.js
- **Python SDK** (`minions-content-publishing`) — Python bindings
- **CLI Tool** (`@minions-content-publishing/cli`) — Command-line interface
- **Web App** — Interactive playground at [content-publishing.minions.wtf](https://content-publishing.minions.wtf)
