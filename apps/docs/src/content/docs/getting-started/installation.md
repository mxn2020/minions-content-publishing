---
title: Installation
description: Install Minions Content-publishing packages
---

## TypeScript / Node.js

```bash
npm install @minions-content-publishing/sdk minions-sdk
```

## Python

```bash
pip install minions-content-publishing
```

## CLI

```bash
npm install -g @minions-content-publishing/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
